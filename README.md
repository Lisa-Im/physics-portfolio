# Physics Portfolio

A personal archive of theoretical physics notes, derivations, and reproducible calculations.

## Website
- https://lisa-im.github.io/physics-portfolio/

## Structure
- `research-notes/` : research-oriented notes 
- `theory-notes/`   : topic/class notes
- `code/`           : notebooks and scripts
- `docs/`           : GitHub Pages website source

## Policies
- No uploading of copyrighted paper PDFs or course material originals.
- Each note/review should include a status tag: draft / checked / validated.
